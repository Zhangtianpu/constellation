# 星座连线游戏 — 需求文档

## 1. 项目概述

### 1.1 项目名称
**Starlight Constellation**（星光星座）

### 1.2 项目愿景
打造一款精美、浪漫、卡通风格的纯前端星座连线游戏，提供两种玩法模式：**经典模式**（2D 平面连线）和**沉浸模式**（第一人称 3D 天球探索）。玩家可以在星空中观察星星，根据提示将星星连线，找到对应的星座。游戏可直接通过 GitHub Pages 部署，无需后端服务。

### 1.3 目标用户
- 对星座和天文感兴趣的休闲玩家
- 喜欢浪漫唯美风格的用户
- 想要学习星座知识的初学者
- 追求放松治愈体验的用户
- 希望模拟真实观星体验的天文爱好者

---

## 2. 功能需求

### 2.1 双模式架构

游戏提供两种核心玩法模式，用户可在主页面自由选择：

| 模式 | 名称 | 核心体验 | 适合人群 |
|------|------|----------|----------|
| 经典模式 | 星座连线 | 2D 平面星空，点击星星连线 | 休闲玩家、初学者 |
| 沉浸模式 | 星空探索 | 第一人称 3D 天球，自由旋转寻找星座 | 天文爱好者、进阶玩家 |

两种模式共享同一套星座数据、评分系统、成就系统和存档系统。

---

### 2.2 经典模式（2D 星座连线）

#### 2.2.1 星空场景
- 页面呈现一片深色夜空背景，带有渐变和微光效果
- 星空中散布若干颗星星（亮点），其中部分星星属于目标星座
- 背景有缓慢闪烁的装饰星星和流星动画，营造浪漫氛围
- 星星有不同大小和亮度，星座内的星星略大于装饰星

#### 2.2.2 星座连线
- 玩家通过点击星星来依次连线
- 连线时出现发光的线条，带有渐变和光晕效果
- 连线过程中，已连接的线段持续发光
- 玩家可以撤销上一步连线（点击撤销按钮或右键）
- 连线顺序需要按照星座的正确连接顺序（从提示的起始星开始）

#### 2.2.3 提示系统
- 游戏开始时显示星座名称和简短描述
- 提供"显示提示"按钮，点击后高亮下一颗需要连接的星星
- 提示使用次数有限，增加挑战性

#### 2.2.4 判定与反馈
- 玩家完成连线后，系统判定是否正确
- 正确时：星座线条变为金色/彩色，星座名称浮现，播放粒子庆祝动画，显示星座故事
- 错误时：错误连线闪烁后消失，提示"再试一次"
- 部分正确时：保留正确部分，提示继续

---

### 2.3 沉浸模式（第一人称星空探索）

#### 2.3.1 核心体验
用户进入页面后，仿佛置身于一片开阔的暗夜平原——脚下是随风微动的碧绿青草，远处环绕着墨色层叠的山峦剪影，头顶是深邃无垠的星空。用户可以通过鼠标拖拽或手指滑动，自由旋转、倾斜、缩放视角，360度环顾四周：仰望天顶的银河，远眺地平线上方的大山轮廓，低头甚至能看见草地与星光的倒影。数千颗星星持续动态闪烁，银河从东北向西南横贯天穹，若隐若现。

#### 2.3.2 天球穹顶系统
- **3D 天球渲染**：所有星星分布在一个巨大的天球内壁上（半径 1000），相机可在距原点 300-1200 的范围内缩放
- **自由旋转**：鼠标拖拽/手指滑动控制视角方向（方位角 + 仰角）
- **缩放**：鼠标滚轮/双指缩放控制相机距离（300-1200），模拟远近观星体验
- **惯性滑动**：松开拖拽后视角缓慢减速，模拟真实观星手感
- **陀螺仪支持**（移动端）：设备倾斜自动旋转视角

#### 2.3.3 地面环境系统 ⭐ v2.0 增强
- **碧绿草地**：脚下铺设一片随风微动的草地平面，草叶使用粒子系统渲染，带有微风摆动动画
- **山峦剪影**：远处环绕多层墨色山峦剪影，由近及远层叠分布，营造纵深感
  - 前景层：近处低矮丘陵，墨色最深，轮廓清晰
  - 中景层：中等高度山脉，略带深蓝色调
  - 远景层：远处高山轮廓，与天空渐变融合
- **星光倒影**：低头看向草地时，可见明亮星星在地面上的微弱倒影，模拟露水反光效果
- **方位标注**：东南西北方位标记，帮助用户定位

#### 2.3.4 星空环境 ⭐ v2.0 增强
- **数千颗星星**：基于真实星表数据（简化版），按赤经/赤纬分布在天球上，所有星座赤纬 > +20°（确保永不落到地面以下）
- **亮度等级**：星星按视星等分为不同大小和亮度（1等星~6等星）
- **动态闪烁**：所有星星持续动态闪烁，低仰角星星闪烁更剧烈，模拟大气折射
- **银河效果**：银河从东北向西南横贯天穹，使用密集星尘粒子 + 淡淡光带渲染，若隐若现
- **大气闪烁**：低仰角星星闪烁更剧烈，模拟大气折射
- **地平线淡出**：低仰角星星根据仰角动态淡出，接近地平线的星星逐渐变暗消失，避免穿模

#### 2.3.5 渐进式提示系统
- **初始提示**：屏幕角落浮现文字提示（如："请寻找北斗七星——它像一把巨大的勺子，位于北方天空，勺口指向北极星"）
- **方位引导**：提示中包含大致方位（"北方天空"、"头顶偏西"等）和形状线索
- **接近反馈**：当用户将视角中心对准星座大致区域时，组成该星座的亮星会发出柔和的光晕并缓慢脉动
- **参考连线**：接近后显示半透明的参考连接线作为引导（仅作引导，不自动连线）
- **提示等级**：
  - Level 1：仅文字描述方位和形状
  - Level 2：星座区域大致高亮，亮星光晕脉动
  - Level 3：显示参考连线和星星脉动

#### 2.3.6 交互连线 ⭐ v2.0 增强
- 用户找到星座后，按正确顺序依次点击脉动的亮星完成连线
- 点击时星星发出光芒涟漪效果
- **连线轨迹以星光粒子效果呈现**：连线不是简单的线条，而是由流动的星光粒子组成的轨迹，粒子沿连线方向缓慢流动，营造星尘飘散的视觉效果
- 连线错误时给出柔和反馈，撤销错误线段
- 连线成功后系统自动比对形状与顺序，判定成功则弹出星图卡片

#### 2.3.7 星座完成奖励 ⭐ v2.0 增强
- **星图卡片**：连线成功后，弹出一张优雅的星图卡片，展示：
  - 星座名称（中英文）
  - 星座连线图案（金色发光线条）
  - 神话故事
  - 最佳观测季节
  - 伴有轻悦的音效
- **星座锁定**：已完成的星座在天球上永久发光显示
- **解锁下一个**：自动解锁下一个星座挑战（如春季的狮子座或冬季的猎户座）
- **天球积累**：随着用户完成更多星座，天球上逐渐形成完整的星座图谱

#### 2.3.8 真实感模拟
- **真实相对位置**：星座在天球上的位置尽可能接近真实天文位置
- **季节关联**：不同季节可见不同星座
- **地平线遮挡**：地平线以下的星星不可见，低仰角星星根据仰角动态淡出
- **透视关系**：山峦、草地的透视关系模拟真实观星场景
- **光污染模拟**（可选）：可调节环境亮度，模拟不同观星条件

### 2.4 关卡系统

#### 2.4.1 关卡设计
- 按难度分级：初级（5-6颗星）、中级（7-9颗星）、高级（10+颗星）
- 每个关卡对应一个真实星座
- 关卡按季节/主题分组：
  - **春夜星空**：狮子座、室女座、牧夫座等
  - **夏夜星空**：天蝎座、天鹰座、天琴座等
  - **秋夜星空**：飞马座、仙女座、英仙座等
  - **冬夜星空**：猎户座、大犬座、双子座等
- 每组包含 4-6 个关卡
- 两种模式共享关卡进度

#### 2.4.2 关卡解锁
- 初始解锁第一组（春夜星空）的第一个星座
- 完成当前星座后解锁下一个
- 完成整组后解锁下一组
- 两种模式中任一模式完成即可解锁

### 2.5 积分与成就

#### 2.5.1 评分机制
- 基础分：完成星座连线获得基础分数
- 速度加成：越快完成，加成越高
- 提示惩罚：每使用一次提示扣减一定分数
- 撤销惩罚：每撤销一次扣减少量分数
- 满分评价：三星评级（★★★）
- 沉浸模式额外加成：因探索难度更高，基础分 ×1.5

#### 2.5.2 成就系统
- **初识星空**：完成第一个星座
- **星座猎人**：完成10个星座
- **星空大师**：完成所有星座
- **完美连线**：不使用提示完成一个星座
- **闪电之手**：30秒内完成一个星座
- **四季守望**：完成所有季节组
- **探索者**：首次在沉浸模式中完成星座
- **天文达人**：在沉浸模式中完成所有星座
- **寻星高手**：在沉浸模式中不使用提示完成星座

### 2.6 游戏界面

#### 2.6.1 主页面
- 浪漫星空背景动画
- 游戏标题（带星光效果）
- **"经典模式"按钮**：进入2D星座连线
- **"沉浸模式"按钮**：进入3D星空探索
- "继续游戏"按钮（有存档时显示）
- 音效开关

#### 2.6.2 关卡选择页
- 季节主题分组展示
- 已完成星座显示金色图标
- 未解锁星座显示为剪影
- 当前星级评价显示
- 模式切换标签（经典/沉浸）

#### 2.6.3 经典模式游戏页面
- 全屏星空场景
- **顶部信息栏（合并）**：返回按钮 | 星座名称 + 描述 | 计时器
- 底部：提示按钮、撤销按钮、重置按钮
- 连线区域：可交互的星空画布

#### 2.6.4 沉浸模式游戏页面
- 全屏 3D 天球场景（含草地、山峦、星空）
- **顶部信息栏**：返回按钮 | 星座名称 + 提示文字 | 计时器
- **方位指示器**：屏幕边缘显示东南西北
- **提示面板**：屏幕角落浮现渐进式提示
- **接近指示器**：视角接近目标区域时，屏幕边缘发出微光
- 底部：提示按钮、撤销按钮、重置按钮

#### 2.6.5 结算页面
- 星座完成动画
- 三星评价展示
- 分数明细
- 星图卡片（星座名称、连线图案、神话故事、最佳观测季节）
- "下一关"和"重玩"按钮
- 模式切换按钮（可在另一模式中重玩）

### 2.7 数据持久化
- 使用 localStorage 保存游戏进度
- 保存内容：已完成关卡、评分记录、成就解锁状态、沉浸模式视角偏好
- 两种模式进度互通
- 支持清除存档重新开始

## 3. 非功能需求

### 3.1 视觉风格
- **整体风格**：浪漫、梦幻、卡通
- **配色方案**：
  - 背景：深蓝到深紫渐变（#0a0a2e → #1a0a3e）
  - 星星：暖白色、淡黄色、淡蓝色
  - 连线：金色/淡蓝色发光线条 + 星光粒子流动效果
  - 草地：深绿到墨绿渐变，带星光倒影
  - 山峦：墨色层叠，由近及远渐淡
  - UI元素：半透明毛玻璃效果
- **动画效果**：
  - 星星闪烁动画（缓慢呼吸式）
  - 流星划过效果
  - 连线绘制动画（星光粒子沿轨迹流动）
  - 完成时的粒子爆炸效果
  - 星云/银河背景缓慢移动
  - 沉浸模式：天球旋转惯性、星星脉动、草地微风摆动、山峦剪影

### 3.2 音效（可选）
- 背景音乐：轻柔的钢琴/竖琴旋律
- 连线音效：清脆的星光音
- 完成音效：庆祝音效
- 错误音效：柔和的提示音
- 沉浸模式：环境音效（虫鸣、风声）

### 3.3 响应式设计
- 支持桌面端（1920×1080 及以上）
- 支持平板端（768×1024）
- 支持手机端（375×667 及以上）
- 触屏和鼠标双支持
- 沉浸模式：支持陀螺仪（移动端）

### 3.4 性能要求
- 首屏加载时间 < 3秒
- 经典模式动画帧率 ≥ 60fps（桌面端）
- 沉浸模式动画帧率 ≥ 30fps（移动端），≥ 60fps（桌面端）
- 总资源大小 < 10MB（含 Three.js 库）
- 天球星星数量：桌面端 3000+，移动端 1000+
- 草地粒子数量：桌面端 5000+，移动端 2000+

### 3.5 兼容性
- Chrome 80+
- Firefox 80+
- Safari 14+
- Edge 80+
- 移动端主流浏览器
- WebGL 支持检测（沉浸模式降级方案）

### 3.6 部署要求
- 纯静态文件，可直接部署到 GitHub Pages
- 无需后端服务
- 无需构建步骤（或构建后为纯静态文件）
- 沉浸模式依赖 Three.js CDN 加载

### 3.7 降级策略
- 不支持 WebGL 的浏览器自动隐藏沉浸模式入口
- 沉浸模式加载失败时提示用户切换经典模式
- 移动端性能不足时自动降低星星数量和特效

## 5. 用户体验流程

### 5.1 经典模式流程
```
主页面 → 选择"经典模式" → 关卡选择页 → 选择星座 → 经典游戏页面
                                                        ↓
                                              阅读提示，开始连线
                                                        ↓
                                              连线完成 → 判定
                                             ↙          ↘
                                          正确           错误
                                           ↓              ↓
                                      结算页面      撤销错误线段
                                           ↓              ↓
                                      下一关/重玩    继续连线
```

### 5.2 沉浸模式流程
```
主页面 → 选择"沉浸模式" → 关卡选择页 → 选择星座 → 沉浸游戏页面
                                                        ↓
                                          阅读方位/形状提示
                                                        ↓
                                          旋转天球，搜索目标区域
                                          （环顾星空、山峦、草地）
                                             ↙              ↘
                                      找到脉动星星       继续搜索
                                           ↓                  ↓
                                    点击亮星连线          使用提示加速
                                    （星光粒子轨迹）
                                           ↓
                                    连线完成 → 判定
                                   ↙          ↘
                                正确           错误
                                 ↓              ↓
                          星图卡片浮现     撤销错误线段
                          （名称+故事+     继续连线
                           连线图案+季节）
                                 ↓
                            下一关/重玩
```
